**–§–æ—Ä–º–∞—Ç –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**

**–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ.** –¢–µ–ª–µ—Ñ–æ–Ω–Ω—ã–π –Ω–æ–º–µ—Ä –¥–æ–ª–∂–µ–Ω —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Ç–æ–ª—å–∫–æ¬†—Ü–∏—Ñ—Ä—ã –∏ —Å–∏–º–≤–æ–ª¬†`-`, –∞ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ü–∏—Ñ—Ä –≤ –∫–∞–∂–¥–æ–π –≥—Ä—É–ø–ø–µ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –ø—Ä–∞–≤–∏–ª—å–Ω—ã–º.

```python

is_valid = True

len = len(str)

for i in range(len):
    part = str[i]
    if (part.isdigit()):
        num = int(part)
        if ((len == 4 and i == 0 and num == 7) or ((len == 3 or (len == 4 and i != 0)) and 0 <= num <= 999) or (i == len - 1 and 0 <= num <= 9999)):
            continue

    is_valid = False
    break

if (is_valid):
    print("YES")
else:
    print("NO")
```
